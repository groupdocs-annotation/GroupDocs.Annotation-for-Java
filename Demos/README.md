# GroupDocs.Annotation for Java Demos

> **Demo only — not for production use.** These sample applications are intended for local development and evaluation. See the [Security Notice](Spring/README.md#security-notice) in each demo README for details.

GroupDocs.Annotation for Java demo projects:

| Demo | Framework | Build |
|------|-----------|-------|
| [Spring](Spring) | Spring Boot 2.0 | `mvn clean spring-boot:run` |
| [Dropwizard](Dropwizard) | Dropwizard 1.3 | `mvn clean compile exec:java` |

Both demos run at `http://localhost:8080/annotation/`.
